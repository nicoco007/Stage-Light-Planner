Welcome to Stage Light Planner!

Stage Light Planner is a simple and easy to use stage planning software.

Getting Started

When you launch the program and create a new plan, this is what you should see:

To start creating a plan, simply drag-and-drop elements from the top bar onto the grid. The gray bar icon is a batten, and the Label allows you to place arbitrary text anywhere on the plan. To edit a fixture, batten, or label's properties, click on it.

Editing fixtures

To create, edit, or delete a fixture, go to Tools > Fixture Editor. There, you can edit or delete existing fixtures, create your own, and sort them by name.

Getting more help

If you need help, start by taking a look at the user manual. If that doesn't answer your question, you can either contact me directly at support@nicoco007.com, or create a new issue over on the project's issue tracker.

About Stage Light Planner
