Welcome to Stage Light Planner!

Stage Light Planner is a simple and easy to use stage planning software.

Getting help

If you need help, start by taking a look at the user manual. If that doesn't answer your question, you can either contact me directly at support@nicoco007.com, or create a new issue over on the project's issue tracker.

About Stage Light Planner

Stage Light Planner is maintained by @nicoco007. It started as a Computer Science project to help with audiovisual technician work at school.
